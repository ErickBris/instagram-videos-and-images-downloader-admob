Installing Requirements - top

As this project use Android Studio, you need to download it first in order to be able to configure the app. Besides Android Studio, you also need to install a Java Development Kit (JDK), at least Version 7 and also Android SDK Tools.
You can download Java Development Kit (JDK) here and Android Studio here. Install JDK first to your computer and after finishing installing JDK, add java path to system variables if necessary. You can read an article about adding java path into system variables here. Now, you can install Android Studio, just install it and double-click android studio icon.
After this you need to donwload Android SDK Tools here.

Opening the project in Android Studio - top

Now, you have already installed all softwares that required to configure Android project. To open Android project to Eclipse follow this steps:

  1. Open Android Studio > File > New > "Import Project"
  2. In the dialog that opens, browse to the project (or a directory containing source files) that you want to import.
  3. Then go to 'Build' Menu > Clean Project
  4. Wait for few minutes until all errors are gone. Ignore any warning appears in 'Problems' tab.

Changing the package name - top

A.    In your Project pane, click on the little gear icon (  )

B.    Uncheck / De-select the Compact Empty Middle Packages option

C.   Your package directory will now be broken up in individual directories

D.   Individually select each directory you want to rename, and:

1.                  Right-click it

2.                  Select Refactor

3.                  Click on Rename

4.                  In the Pop-up dialog, click on Rename Package instead of Rename Directory

5.                  Enter the new name and hit Refactor

6.                  Allow a minute to let Android Studio update all changes

7.                  Note: When renaming com in Android Studio, it might give a warning. In such case, select Rename All

E.    Now open your Gradle Build File (build.gradle - Usually app or mobile). Update the applicationId to your new Package Name and Sync Gradle, if it hasn't already been updated automatically

 

Changing app name - top

  1. Open Andorid Studio > App > res > values > strings.xml
  2. Select 'strings.xml' tab next to 'Resources' tab
  3. Enter your app name inside "app_name" string tag:
    <string name="app_name">Instagram Downloader</string>
This will change both title inside the app & app icon name.

Setting Up Admob - top

To set up Admob Ads Go to apps.admob.com and login with your Admob account, if you don't know how to monetize a new app please see this guide here from Google about how to do it. Below are steps to set up Admob on Android project:

  1. Open Andorid Studio > App > res > values > strings.xml
  2. Select 'strings.xml' tab next to 'Resources' tab
  3. Enter your AdMob Banner id inside "banner_ad_id" string tag:
    <string name="banner_ad_id">Your AdMob Banner ID Here</string>
  4. Enter your AdMob Interstitial id inside "Interstitial_ads_id" string tag:
    <string name="Interstitial_ads_id">Your AdMob Interstitial ID Here</string>

Updating App Version - top

When you want to update your app on Google Play, make sure that you change the Version code and Version name on AndroidManifest.xml file that is located under res folder, to higher number than previous version and save.
